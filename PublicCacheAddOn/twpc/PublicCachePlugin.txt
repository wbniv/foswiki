---+ Public Cache Plugin

Support Plugin for the PublicCacheAddOn, in charge of invalidating its
cache on topic saves.

---++ Plugin Installation Instructions

   * This plugin is automatically installed by the PublicCacheAddOn, you do not need to install it.
