---+!! !TwistyPlugin Bundle for TWiki 4.1.x

This is a convenience package for installing !TwistyPlugin on TWiki 4.1 and TWiki 4.1.2.
Installed extensions are:
   * http://TWiki.org/cgi-bin/view/Plugins/BehaviourContrib
   * http://TWiki.org/cgi-bin/view/Plugins/TwistyContrib
   * http://TWiki.org/cgi-bin/view/Plugins/TwistyPlugin

Other Twisty bundles:
   *  http://TWiki.org/cgi-bin/view/Plugins/TwistyBundleTWiki04x00x05 for TWiki 4.0.5
   *  http://TWiki.org/cgi-bin/view/Plugins/TwistyBundleTWiki04x02 for TWiki 4.2
   
---++ Installation Instructions
   1 Run the installer to install all files
   1 Go to =configure= to enable !TwistyPlugin

---++ Contrib Settings
   * Set SHORTDESCRIPTION = Contains all necessary files to use !TwistyPlugin on TWiki 4.1 and 4.1.2
