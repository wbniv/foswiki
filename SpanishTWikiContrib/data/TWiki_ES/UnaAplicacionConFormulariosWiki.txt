---++ Pregunta:
Cómo puedo crear una aplicación sencilla basada en FormulariosTWiki?


---++ Respuesta:

   1 Cree un formulario TWiki y defina campos (ver FormulariosTWiki para más detalles)
   1 Agregue el formulario a la lista de formularios de la web. Definir la variable WEBFORMS en WebPreferences %FLAGGB%
   1 Cree una plantilla de página
   1 En modo de edición, agregue el [[FormulariosTWiki][formulario TWiki]] que definió para la plantilla de página utilizando el botón "Agregar formulario"
   1 Cree un formulario HTML con los campos necesarios
   1 Defina la función =SEARCH= para listar los items (ver %SYSTEMWEB%.VarSEARCH para más detalles)

Para más información siga los enlaces...

A continuación el código de PreguntasFrecuentes:

<verbatim>
<form action="%SCRIPTURL{"edit"}%/%WEB%/">
Nueva página de preguntas frecuentes: &nbsp; (Utilice una PalabraWiki como nombre) <br />
<input type="text" name="topic" size="32" /> &nbsp; <input type="submit" class="twikiSubmit" value="Crear página" />
<input type="hidden" name="onlywikiname" value="on" />
<input type="hidden" name="templatetopic" value="PlantillaTWikiFaq" />
<input type="hidden" name="topicparent" value="%TOPIC%" />
</form>
</verbatim>
