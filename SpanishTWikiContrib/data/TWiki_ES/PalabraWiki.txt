---+!! Qué es una !PalabraWiki?
 Una !PalabraWiki consiste de dos o más palabras pegadas que empiezan con una letra mayúscula. *PalabrasWiki son títulos de una páginaë*. El título de una página TWiki siempre tiene un formato fijo: dos o más palabras pegadas que empiezan con una letra mayúscula. Igual como por ejemplo el título de esta página: *PalabraWiki*. Cuando digite el nombre de una página, crea un enlace a esta página. Puede digitar =%HOMETOPIC%= y cuando salve la página, se convierte en %HOMETOPIC%. Tan fácil es.

Enlaces de !PalabrasWiki son fáciles de usar:
   * Uno no tiene que saber la ruta completa donde se guardó una página - nada más se digita el nombre
   * Uno no tiene que escribir en HTML
   * Sin HTML, el texto de la página es más fácil de leer en el momento de editar una página
   * Enlaces fáciles favorecen la generación de textos interesantes con enlaces ubicados en el contexto

!PalabrasWiki tienen el formato que tienen porque:
   * Enlaces serán reconocidos instantaneamente
   * Evita la necesidad de tener que manejar comandos de HTML
   * Evita títulos de páginas que no dicen nada porque se requiere mínimo dos palabras.

---++ Sintaxis de !PalabrasWiki
   * Letra(s) en mayúscula
   * Letra(s) en minúscula o número(s)
   * Letra(s) en mayúscula
   * Letras opcionales en minúscula o mayúscula o número(s)

---++ Buenos ejemplos de !PalabrasWiki
   * PalabraWiki
   * BuenEstilo
   * VersionFiveDotThree
   * UnaPalabraWikiLargaTambienEsPosible: donde se permiten letras en mayúscula o minúscula, un grupo de letras del mismo tipo está permitido también.
   * Por1Hora: Acuérdese que números son considerados como letras en minúscula dentro de !PalabrasWiki

---++ Malos ejemplos de !PalabrasWiki:
   * Web: Nombre sin la secuencia de letra en mayúscula, letra en minúscula, letra en mayúscula
   * 5TWiki: Nombre empieza con un número
   * Know-How: Nombre con guión en el centro

---++ Variaciones de enlaces

Cuando escriba el título de una página, se convierte en un enlace. Existen varias formas

   * Para escribir una *etiqueta personalizada* para un enlace, utilice corchetes: =[<nop>[CómoFormatearTexto][Reglas básicas para la introducción de texto]]= - este se convierte en: [[CómoFormatearTexto][Reglas básicas para la introducción de texto]]
   * Para crear un enlace a una página que se encuentra *en una red diferente*, escriba: =Sandbox.WebSearch= - este se convierte en: Sandbox.WebSearch
   * Para mostrar el nombre de la red dentro del enlace, utilice corchetes: =[<nop>[Sandbox.%HOMETOPIC%]]= - este se convierte en: [[Sandbox.%HOMETOPIC%]]
   * Para crear un enlace a *otra página Wiki*, utilice: =TWiki:Main/WebHome= - este se convierte en: TWiki:Main/WebHome (sitios están definidos en el TWiki.InterwikiPlugin %FLAGGB%)
   * Para crear un enlace a una parte que está *en la misma página*, escriba "#" seguido por el nombre de una palabra ancla. La palabra ancla es "#" seguido por una palabra que tiene que ser una !PalabraWiki. Ejemplo: =#MiAncla=. También puede enlazar a una ancla en otra página: =%TWIKIWEB%.%HOMETOPIC%#MiAncla=.
   * Para crear un enlace a un título que está *en la misma página*, escriba "#" seguido por el texto del título, reemplazando los espacios por lineas bajas (y sin =!=): =[<nop>[#Buenos_ejemplos_de_PalabrasWiki]]= se convierte en: [[#Buenos_ejemplos_de_PalabrasWiki]]. También puede enlazar el título de otra página: =%TWIKIWEB%.%HOMETOPIC%#Disclaimer= se convierte en: %TWIKIWEB%.%HOMETOPIC%#Disclaimer.

---++ Consejos
   * Utilice !PalabrasWiki cuando pueda. Muchos enlaces ayudan a convertir un Wiki en un éxito.
   * Sea preciso. Todas las páginas comparten el mismo sitio. Por ejemplo, en lugar de =ManualEspecial= escriba =ManualEspecialParaCucharas= porque puede ser que otro proyecto tenga manuales especiales también.
   * Para evitar que una !PalabraWiki sea convertida a un enlace, agregue un signo de exclamacion inmediatamente antes de la !PalabraWiki. Por ejemplo, escriba =!<nop>SunOS= para obtener !SunOS.
   * A veces tiene que ser creativa para encontrar una buena !PalabraWiki. Por ejemplo: 
      * Para crear una página para el producto *Cuchara especial 1.2*, utilice =CucharaEspecial1punto2= o =CucharaEspecial1pt2=, pero no =CucharaEspecial1.2=.
      * Números se consideran como minúscula, lo cual convierte =Año2Mil= en una !PalabraWiki, pero no =Año2000=.
      * Convierte acrónimos en !PalabrasWiki, por ejemplo utilice =FaqIndex= para la página "FAQ index".
   * Es posible de apagar la generación automática de !PalabrasWiki y basarse unicamente en la anotación con corchetes. Revise el comando NOAUTOLINK en TWiki.TWikiPreferences#Default_Web_Preferences %FLAGGB%.
   * %X% Cuando cree un enlace a la página <nop>%HOMETOPIC% _en otra red_, el enlace se va a mostrar como el nombre de la red, por ejemplo =Sandbox.%HOMETOPIC%= se convierte en Sandbox.%HOMETOPIC%.
