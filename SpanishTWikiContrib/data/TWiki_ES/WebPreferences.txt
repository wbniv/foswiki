---+!! <nop>%WEB% Web Preferences

Las configuraciones a continuación son __preferencias de web__ de la web [[%WEB%.%HOMETOPIC%][%WEB%]]. Estas preferencias sobrescriben las __preferencias del sitio__ en [[%SYSTEMWEB%.%WIKIPREFSTOPIC%]] y [[%LOCALSITEPREFS%]], y pueden ser sobrescritos por __preferencias de usuarios__ (su tema personal, por ejemplo: %USERSWEB%.%WIKINAME% en la web [[%USERSWEB%.%HOMETOPIC%][%USERSWEB%]]).

---++ Configuración de preferencias de web
Estas configuraciones sobrescriben los valores estándares solamente para esta web. Ver [[%SYSTEMWEB%.DefaultPreferences#DefaultWebPreferences][lista completa de valores estándares con explicación]].

   * Configuraciones de web:
      * Set WEBTOPICLIST = [[BienvenidoHuesped][Bienvenido]] %SEP% [[TWiki.TWikiRegistration][Registro]] %SEP% [[WebChanges][Cambios]] %SEP% [[WebTopicList][Temas]] %SEP% [[WebIndex][Indice]] %SEP% [[WebSearch][Búsqueda]] %SEP% Ir a <input type="text" name="topic" size="16" />
      * Set WEBBGCOLOR = #99CC99
      * Set WEBHEADERART = %PUBURLPATH%/%SYSTEMWEB%/%WEBPREFSTOPIC%/Dakar.gif
      * Set SITEMAPLIST = on
      * Set SITEMAPWHAT = Traducción de documentación TWiki
      * Set SITEMAPUSETO = ...proveer documentación en español para aprender cómo manipular el TWiki
      * Ocultar tabla de contenido si tema está incluido
         * Set TOC_HIDE_IF_INCLUDED = on
      * Definir título estándar de tablas de contenido
         * Set TOC_TITLE = En esta página:
      * El primer nivel de títulos a aparecer en la tabla de contenido:
         * Set TOC_MIN_DEPTH = 2

   * Plantilla estándar para *temas nuevos* de esta web:
      * WebTopicEditTemplate: Site-level default topic template

   * Lista de *formularios* separados por como que pueden ser añadidos a temas de esta web. Ver %SYSTEMWEB%.TWikiForms para más información.
      * Set WEBFORMS = %SYSTEMWEB%.TranslationForm

      * Set ALLOWWEBCHANGE = %USERSWEB%.TWikiAdminGroup
      * Set ALLOWWEBRENAME = %USERSWEB%.TWikiAdminGroup

   * Usuarios o grupos quienes tienen permiso de modificar o renombrar este tema <nop>%TOPIC%: (e.g., %USERSWEB%.WebAdministratorsGroup)
