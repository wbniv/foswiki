_El sitio para conocer las características de TWiki y para realizar el mantenimiento del sistema TWiki._

Este sitio está utilizando la versión *%WIKIVERSION%*, Plugin API versión *%PLUGINVERSION{}%*

TWiki es una plataforma de colaboración flexible, poderosa, segura y aún simple basada en una red. Utilice TWiki para efectuar el desarrollo de un proyecto, un sistema de administración de documentos, una base de conocimiento o cualquier otra herramienta tipo groupware sea en un intranet o en la internet.
