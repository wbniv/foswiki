#VarSERVERTIME2
---+++ SERVERTIME{"format"} -- formatted server time
   * Same format qualifiers as =%<nop>GMTIME%=
   * Syntax: =%<nop>SERVERTIME{"format"}%=
   * Example: =%<nop>SERVERTIME{"$hou:$min"}%= expands to =%SERVERTIME{"$hou:$min"}%=
   * __%X% Note:__ When used in a template topic, this variable will be expanded when the template is used to create a new topic. See TemplateTopics#TemplateTopicsVars for details.
