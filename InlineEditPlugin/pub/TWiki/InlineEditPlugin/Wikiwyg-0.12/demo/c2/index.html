Using Greasemonkey, I was able to show Ward Cunningham, the father of wiki, what his famous c2.com wiki would look like with Wikiwyg. Ward liked the idea, and is looking forward to installing it for real in the near future.

You can try it at home:

  1. Use Firefox
  2. Install Greasemonkey 0.5
  3. Install this user script
  4. Visit Ward's Wiki
  5. Doubleclick on the page

Note: You can't save changes yet. This is just a proof of concept spike.
