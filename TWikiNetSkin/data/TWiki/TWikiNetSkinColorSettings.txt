---+ !!Color scheme settings for TWikiNetSkin

   * Set SKIN = twikinet,pattern

---++!! Usage
---+++!! Step 1
Let %SYSTEMWEB%.TWikiNetSkin point to the custom-made colors style sheet. In [[%MAINWEB%.TWikiPreferences]] put:
<verbatim>
   * Set USERCOLORSURL = %PUBURLPATH%/%SYSTEMWEB%/TWikiNetSkin/theme-colors.css
</verbatim>

---+++!! Step 2
Edit this topic, and change the color variables as you like. 

---+++!! Step 3
Save the topic. This will generate an [[%PUBURLPATH%/%SYSTEMWEB%/TWikiNetSkin/theme-colors.css][attachment]] to topic [[TWikiNetSkin]].
